# ARTI308 | Lab 3: Exploratory Data Analysis (EDA)

## Objective
Perform exploratory data analysis on a car price dataset to gain insights into
the data structure, identify missing values, analyze distributions, and study
relationships between features.

## Dataset
- Car price dataset including attributes such as brand, model year, mileage,
engine size, fuel type, transmission, condition, and price.

## Notebook Overview
The notebook covers the following steps:
- Loading and inspecting the dataset
- Identifying missing and duplicate values
- Examining dataset shape and column data types
- Generating summary statistics
- Univariate analysis of key features
- Bivariate analysis (Mileage vs Price)
- Correlation matrix visualization and interpretation

## Tools & Libraries
- Python
- pandas
- matplotlib
- seaborn
