# ARTI 308 – Machine Learning  
## Lab 2

### Objective
The objective of this lab is to identify a machine learning problem, select a suitable open dataset, and understand the basic steps of a machine learning workflow. The lab also aims to practice loading and inspecting a dataset using Python and visualizing the methodology using a diagram.

---

### Folder Contents
- `lab2.ipynb` – Jupyter Notebook used to load the dataset and display basic information such as shape, columns, data types, and missing values.  
- `Dataset.csv` – The selected open dataset used in this lab.  
- `methodology_diagram.png` – Diagram illustrating the machine learning workflow from data selection to results.  
- `README.md` – Summary of the lab objective and repository contents.
